# Language Screen Input

## Target File

`app/src/main/java/com/project/composeproject/ui/screen/language/LanguageScreen.kt`

## User Request

Dựng màn Language theo ảnh thiết kế được cung cấp.

## Design Summary

- Background màu xám rất nhạt màu #f3f4f6.
- Top bar cao khoảng 42dp, có padding status bar.
- Tiêu đề `Language` bên trái, chữ đậm màu đen.
- Nút `DONE` bên phải, có icon check phía trước chữ.
- Danh sách ngôn ngữ dạng các pill bo tròn full width.
- Item đầu tiên `English` đang được chọn.
- Item được chọn có border đen mảnh, nền trắng, radio selected màu đen, text đậm.
- Item chưa chọn nền trắng, không border, radio unselected viền xám, text đậm.
- Khoảng cách item dọc khoảng 10dp.
- Padding ngang màn hình khoảng 8dp.
- Danh sách gồm: English, German, Arabic, Hindi, Spanish, Mandarin, French.
- Text màu #000000
